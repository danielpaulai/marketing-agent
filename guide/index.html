Install your Marketing Agent

Five skills, one voice file, zero API keys. Ten minutes from here to an agent that writes like you. Measured once, fits forever.

BEFORE YOU START

Two things you need

1. A paid Claude plan (Pro or Max). Claude Code comes built into the desktop app on paid plans. The free plan does not include it, so upgrade first at claude.ai or you will only see the chat.

2. The Claude Desktop app. Download it at claude.ai/download, install it like any normal app, and sign in. Claude Code is already inside, there is nothing extra to set up.

That is everything. No terminal, no developer tools, no API keys, no separate Claude Code install.

STEP 1

Install the plugin (three clicks, no terminal)

Open the Claude Desktop app, then:

1. Open Settings, then Customize, and find Plugins.
2. Choose Add marketplace (from a repository) and paste this URL:

https://github.com/danielpaulai/marketing-agent

3. Find marketing-agent in the list and click Install.

Done. The app pulls the plugin straight from the repository. No API keys, no connectors, nothing to configure.

Prefer zero installs? Download the ZIP from github.com/danielpaulai/marketing-agent (green Code button, Download ZIP), unzip it, and open that folder in Claude Code. The skills work right there, no install at all.

STEP 2

Get measured (once)

In the Claude Code desktop app, start a new session and choose a folder for your agent to live in. Make a new folder called my-marketing-agent when it asks where to work. This folder is your agent's home, everything it learns lives here.

Then say this, and attach or paste your material:

Extract my voice. Here are 5 posts I wrote, a transcript of me talking, and a few client replies.

The agent interviews you, analyzes your real writing, and saves VOICE-DNA.md in the folder. That file is the ledger. Every skill reads it first, so you never answer voice questions again.

Bring real material: 5+ posts you wrote yourself, one voice note or call transcript, a few DM or email replies. The messier and more human, the better the extraction.

STEP 4

Pull the trigger, three ways

1 · THE COMMAND

Type the request yourself. You pull the trigger. Great for tonight.

2 · THE ASK

Just describe what you need: "I need a post for tomorrow." Claude reads every skill's description and fires the right one itself.

3 · THE CLOCK

The standing order. Say it once, it runs forever, whether you show up or not. This is what makes it an agent.

Set your standing order. In Claude Code, say:

Create a routine: every morning at 7am, look at my business in VOICE-DNA.md, find one fresh, on-trend content idea for it, write the post in my voice using the write-post skill, and leave the post and an image prompt for it in a file called today-post.md.

You wake up, open today-post.md, read it, post it. Done before coffee.

TROUBLE?

Quick fixes

I cannot find Plugins in the app

Look under Settings, then Customize, then Plugins. If you do not see it, you are likely on the free plan (Claude Code needs Pro or Max) or the app needs an update. In a terminal session of Claude Code, typing /plugin opens the same manager.

The plugin install says marketplace not found

Check the spelling: danielpaulai/marketing-agent. You need an internet connection, and the app should be up to date.

My posts do not sound like me

The ledger needs better measurements. Say "tune my Voice DNA" and tell the agent exactly what feels off, or rerun extract-voice with more real samples, especially a talking transcript.

Where do the images come from?

The image-post and carousel skills give you finished prompts, you paste them into any image tool you already use (gpt-image, Midjourney, and similar). No image account is required to use everything else.
